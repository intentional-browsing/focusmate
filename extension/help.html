Help

To create a new task, press the “Create Task” button. Each task requires a name, trigger, and action.

Triggers

A trigger is used to execute an action when a certain condition is met. There are three types of triggers:

  • A site is opened: execute an action based on a website you open
    • Host name: the website URL
  • Set Time of Day: execute an action based on a specific time. Must be written in military time
    • Hour of Day: must be between 0 and 23
    • Minute of Hour: must be between 0 and 59
  • Set a Timer: execute an action after a set period of time
    • Hours: must be between 0 and 23
    • Minutes: must be between 0 and 59
    • Seconds: must be between 1 and 59

Actions

An action executes commands like closing a website to giving you a reminder. There are four types of actions:

  • Close Site: closes the website. Can only be used with A site is opened.
  • Close and then Open Another Site: closes the site and opens a different site based on the provided URL. Can only be used with A site is opened.
    • Host name: the website URL to open
  • Open a website: opens a website based on the provided URL
    • Host name: the website URL to open
  • Set a reminder: opens a new tab, showing a reminder with a message you provided
    • Message: message for reminder to present

Popup

When you click the icon on the extensions bar, a small popup with three buttons appears. You can set a timer, set reminder, or view your tasks in this popup.
