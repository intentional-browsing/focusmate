About

Intentional Browsing is a Google Chrome extension that is aimed to help you exercise more control over your browsing. This extension helps you set up simple if-then conditions. For instance, you can set up rules like 'If a tab of youtube.com is opened, close it'.

Remember to pin Intentional Browsing to the Chrome Toolbar!
Click on the extension icon in the top left corner of Chrome, and click on the pushpin icon next to Intentional Browsing.

Getting Started
Check out the Tasks page. It can be accessed by clicking on the Intentional Browing icon and clicking 'View Tasks'. There are tasks already added for you, but you can delete them and create your own!
